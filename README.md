# Fine-tuned GPT-2 Model

# ⚠️Note: Please run only on Google colab

## 📋 Project Overview
- My purpose in this project is to understand how LLM works.
- It involves fine-tunined OpenAI's GPT-2 model on New data.
- The model has been trained using Google Colab's GPU resources.

**Key Components:**
- **Transformer Decoder Blocks**: GPT-2 uses only the decoder part of the transformer architecture
- **Multi-Head Self-Attention**: Allows the model to focus on different parts of the input text
- **Feed-Forward Networks**: Process the attention outputs
- **Positional Embeddings**: Help the model understand word order since transformers aren't sequential
